HTTP Protocols

HTTP stands for Hyper Text Transfer Protocol. It is the language used to communicate between web browsers and servers. HTTP is used to transfer data from a server to a browser to view web pages. HTTP is also used to transfer images, text and sound.

HTTPS Protocols

HTTPS stands for Hyper Text Transfer Protocol Secure. It is the more secure version of HTTP and the protocol over which data is sent between your browser and the website you’re connected to. The “Secure” portion of HTTPS refers to an encryption on this communication. HTTPS is used to protect confidential transactions like online banking/shopping.
